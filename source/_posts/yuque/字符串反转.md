或者遍历到 StringBuffer

```shell
    public String solve (String str) {
        // write code here
        if (str.length() == 0) {
            return "";
        }
        char c[] = str.toCharArray();
        StringBuffer s = new StringBuffer();
        for (int i = c.length-1; i >= 0; i--) {
            s.append(c[i]);
        }
        return s.toString();
    }
```
